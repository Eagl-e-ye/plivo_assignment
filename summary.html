Turn Detection Speedrun Submission

Problem

Predict whether a pause represents an End-of-Turn (EOT) or a Hold using only causal information before the pause.

Approach

  • Handcrafted prosodic features
  • Energy statistics
  • Pitch statistics
  • Voicing statistics
  • MFCC statistics
  • Spectral features
  • Extra Trees classifier
  • MFCC-based CNN classifier

Dataset

  • English
  • Hindi
  • 496 total samples

Results

Model Validation AUC
Extra Trees 0.667
CNN 0.743

Future Improvements

  • CNN + Extra Trees ensemble
  • Feature selection using importance ranking
  • Hyperparameter tuning
  • Threshold optimization using score.py
